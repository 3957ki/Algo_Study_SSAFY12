이번엔 단순하게 반복문 구현으로 이동 가능 지역을 탐색했고, 어려운 점은 없었다

## 2번
- BFS 탐색으로 가장 멀리 떨어진 2지점 사이의 거리를 구했다
- BFS에서 사용되는 Node에 거리 값을 가지고 이동하도록 했다

Node에 값을 가지고 이동하지않고, level별 BFS를 사용했다면 조금도 효율성이 좋다고 생각했다.
