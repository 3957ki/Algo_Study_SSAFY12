### 2번

문제의 입력으로 주어지는 정수 값의 범위를, 탑의 높이의 정수 값의 범위로 착각해서
히든 케이스를 통과하지 못했다.
입력 값의 기준을 헷갈리지 않도록 조심해야겠다!
시작 값을 1~1000으로 주어 탐색을 했는데, 다른 코드를 보면 index를 이동시켜가면서 탐색하는 경우도 있었다.
경우에 따라 효율적인 방법을 사용할 수 있도록 해야겠다.
